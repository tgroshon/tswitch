TSwitch sliding switches for the web

 

 

Requirements

  • jQuery
  • jQuery UI
  • TSwitch JS Widget (included)
  • TSwitch Images (included)

Usage

Without Options

$("#tswitcher").tswitch()
With Options

$("#tswitcher").tswitch({imagePath: "/img/", flippedOn: true})

Options

imagePath

A path to the twsitch images.

flippedOn

Starting state of the switch

size

"default" or "other"

Methods

.tswitch("flip")

Manually toggles the switch to the opposite state

$('#tswitcher').tswitch("flip")

.tswitch("switchOn")

Manually set switch to on position

$('#tswitcher').tswitch("switchOn")

.tswitch("switchOff")

Manually set the switch to off position

$('#tswitcher').tswitch("switchOff")

.tswitch("destroy")

Removes switch elements from container

$('#tswitcher').tswitch("destroy")

Events & Callbacks

tswitchflippedon

Fired on click event, when switch set to 'on' position

$('#tswitcher').on('tswitchflippedon', function(event){ ... });

tswitchflippedoff

Fired on click event, when switch set to 'off' position

$('#tswitcher').on('tswitchflippedoff', function(event){ ... });

tswitchoncomplete

Fired after every switch from 'on' position

$('#tswitcher').on('tswitchoncomplete', function(event){ ... });

tswitchoffcomplete

Fired after every switch to 'off' position

$('#tswitcher').on('tswitchoffcomplete', function(event){ ... });

*Chaining of callbacks is supported

Other Features
